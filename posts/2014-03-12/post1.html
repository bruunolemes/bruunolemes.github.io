Github + HTML + CSS

19/03/2014 Bruno Lemes

Nesta aula, após criarmos o cadastro no site github.com, começamos a desenvolver uma página modelo, uma espécie de blog sobre determinado assunto à escolha. Iniciamos comentando um pouco sobre determinadas formas de criarmos uma página inicial (index.html) do tipo html, linguagem utilizada para desenvolvimento de páginas web. Após a criação desta index, usando por enquanto o famoso "bloco de notas" do próprio windows e salvando com a extensão .html, partimos então para o desenvolvimento da página por si própria, criando as tags, que são referências usadas para a montagem do layout da página.

Após a criação das tags básicas do html juntamente ao cabeçalho oficial obrigatória, cuja função é fazer com que os navegadores, os browsers sejam eles móveis ou não entenda o que está sendo aberto, no caso a página html criada. As tags básicas são o html,head,body,h1,p. Após a criação do corpo da página vimos para que serve cada uma das tags criadas e que conteúdo colocar dentro de cada uma. Após isso, vimos um pouco sobre estilos, que são os tipos de formatação que podemos fazer com o conteúdo inserido dentro das tags, aprendemos então a utilizar uma nova tag chamada style.
